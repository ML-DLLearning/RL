# Reinforcement Learning
**核心要素：**  
环境（Environment）：智能体所处的世界。  
状态（State, s）：当前环境对智能体可见的信息。  
动作（Action, a）：智能体可执行的操作。  
奖励（Reward, r）：环境对动作好坏的即时反馈，可能稀疏或延迟。  
策略（Policy, π）：从状态到动作的映射，决定如何行动。  
价值函数（Value/Q）：评估某状态或状态-动作在长期回报上的好坏。  

**基本交互循环：**  
观察状态 s  
按策略 π 选择动作 a  
环境转移到新状态 s' 并给出奖励 r  
用 (s, a, r, s') 更新价值估计或直接更新策略  
重复，目标是最大化长期累计奖励
